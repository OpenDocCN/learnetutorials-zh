## 下载

### Docker

```
docker pull apachecn0/learnetutorials-zh
docker run -tid -p <port>:80 apachecn0/learnetutorials-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g learnetutorials-zh
learnetutorials-zh <port>
# 访问 http://localhost:{port} 查看文档
```
